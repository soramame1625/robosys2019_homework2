# ロボットシステム学2019 課題2
- ジャンクの二足歩行ロボットを購入し、修理後コントロールボードをRaspberryPiに換装。ROSによるコントロールを可能にした。
- Gazeboの逆運動学ライブラリを利用し歩行モーション作成
- joystickdriversを利用しPS4専用コントローラ（DualShock4）により操作可能
- xero_cntl.launchを実行で起動
