# ロボットシステム学2019 課題2
-ジャンクの二足歩行ロボットを購入し、コントロールボードをRaspberryPiに換装。ROSによるコントロールを可能にした。
-Gazeboの逆運動学ライブラリを利用し歩行モーション作成
-joystick-driversを利用しPS4専用コントローラ（DualShock4）により操作可能
